# TPO-ParkingAlert Backend

This is the backend for the **TPO-ParkingAlert** project, built using **Django** and **Django Rest Framework**.

## Tech Stack
- Python
- Django
- Django Rest Framework
- PostgreSQL

## Description
The backend provides REST API endpoints for managing users and parking-related data such as:
- Slovenska mesta (cities)
- Slovenske ulice (streets)
- Parkirna mesta (parking spots)

It supports basic CRUD operations (GET, POST, PUT, DELETE) and is designed to be consumed by a mobile or web frontend.
